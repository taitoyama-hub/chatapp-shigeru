# ①課題名
チャットアプリ　相互返信版

## ②課題内容（どんな作品か）
- Firebase Realtime Databaseを使用したチャットアプリです。
- 授業の内容に加え、送信すると自動的に「シゲル」という相手から返信が来る機能を実装しました。
- 吹き出し表示で、自分のメッセージは左側、シゲルのメッセージは右側に表示されます。
- 各メッセージには名前と投稿日時が小さく表示され、たまにシゲルが貼るURLはクリック可能なリンクとして表示されます。

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ④アプリのログイン用IDまたはPassword（ある場合）
なし

## ⑤工夫した点・こだわった点
- **LINE風の吹き出し表示**: 自分のメッセージは左側に灰色、シゲルのメッセージは右側に青色の吹き出しで表示し、分かりやすく。
- **自動返信機能**: メッセージを送信すると1秒後に「シゲル」から自動的に返信が来る機能を実装しました。返信は「いいんちゃう」「しらんけど」「これみとき」の3つからランダムに選択（先日学習したrandam.math関数を使用）。いいんちゃうの時はif分岐でURLを貼り付け。
- **URL自動リンク化**: メッセージ内のURLを自動検出し、クリック可能なリンクに変換する機能を実装しました。
- **2カラムレイアウト**: 入力欄を左側にコンパクトに配置し、表示欄を右側に大きく配置することで、使いやすいUIを意図。
